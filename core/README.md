В этом модуле собраны все независимые классы, доступ к которым нужен во всех 3х модулях, чтобы избавиться от дубля

> Например: AABB самостоятельный класс который не зависит от других, его использует renderer, client, server
> чтобы не дублировать класс и не делать сложных преобразований он лежит в этом модуле
